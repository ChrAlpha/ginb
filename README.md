# ginb

GitHub Issues Next.js Blog

## 介绍 Introduction

ginb (GitHub Issues Next.js Blog) 是一个基于 GitHub Issues 作为 CMS、Next.js 作为前端框架的博客模板。

## 使用 Usage

1. Fork 本仓库
2. 修改 `_config.js` 文件中的 `sitename` 和 `description` 字段，有意绑定域名的亦需修改 `url`
3. 在 Vercel 关联你的仓库并部署

更多配置请参考 Wiki。
